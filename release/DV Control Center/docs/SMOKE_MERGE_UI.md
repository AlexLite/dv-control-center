# Smoke: Merge Scale (Flex Studio)

Цель: быстро проверить, что `Merge Engine` меняет `Scale` между двумя пресетами, а не только `X/Y`.

## Preconditions

- UI открыт на `http://127.0.0.1:9999`.
- Вкладка `Flex Studio`.
- Панель `Merge Engine` раскрыта (`✨ Merge`).

## Steps

1. Установить `flex1-s = 100`, сохранить пресет `SMOKE_A`.
2. Установить `flex1-s = 75`, сохранить пресет `SMOKE_B`.
3. Нажать `Run` для `SMOKE_A`, дождаться конца анимации.
4. Проверить, что `flex1-s` стал `100.00`.
5. Нажать `Run` для `SMOKE_B`, дождаться конца анимации.
6. Проверить, что `flex1-s` стал `75.00`.

## Expected Result

- Во время оффлайн режима статус содержит `Preview only`.
- Значение `Scale` реально меняется между `100.00` и `75.00`.
- Пресеты присутствуют в списке и запускаются без ошибок.
